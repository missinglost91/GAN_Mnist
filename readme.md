# GAN 模型实战

## 项目介绍 
基于生成对抗网络，以MNIST数据集为例，实现生成手写数字图片的功能。

[视频链接](https://www.bilibili.com/video/BV1rUqUBeEkc?t=52.2)

该项目由以下几个文件组成：
- `gan.py` 主体代码
- `config.yaml` 配置文件
- `requirements.txt` 依赖库文件
- `dataSet/` 数据集文件夹

## 

在终端打开GAN文件夹，确保已经激活了虚拟环境


执行以下命令安装依赖库：

```bash
pip install -r requirements.txt
```

可以在配置文件`config.yaml`中修改参数

记得cirl+s保存配置文件

```bash
python gan.py
```
